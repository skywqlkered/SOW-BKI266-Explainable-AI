## Install
To install the necessary files to grade/test this project:
```bash
pip install -r requirements.txt
```

Run the download script, this downloads the dataset and moves it to the current working directory:
```bash
py download_dataset.py
```
Note: If the file moving for whatever reason doesn't come through, the path of the dataset is printed in the console. The folder `network-intrusion-detection` including its content should be moved to the repository folder as seen in the file structure.
